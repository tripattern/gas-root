# gas-root

## Clasp
```
npm install @google/clasp --no-save
./node_modules/.bin/clasp login
# Default credentials saved to: ~/.clasprc.json
```
### Create a project
```
mkdir project-name
cd project-name
../node_modules/.bin/clasp create --title "Project Name"
```

### Clone Project
```
../node_modules/.bin/clasp clone <scriptID>
```
